# Research Papers Folder

Place your research papers (PDF files) here.

## How to Add Research Papers

1. **Save your PDF files** in this folder
2. **Name them clearly** (e.g., `climate-financing.pdf`, `digital-planning.pdf`)
3. **Update the paths** in `src/data/researchItems.ts` to match your filenames

## Current Research Papers Expected:
- climate-financing.pdf
- digital-planning.pdf
- critical-minerals.pdf

## Example:
If you have a paper called "Climate Finance Report 2024.pdf":
1. Copy it to this folder
2. Rename to: `climate-financing.pdf`
3. The "View Research Paper" link will automatically work!

## File Guidelines:
- **Format:** PDF only
- **Size:** Keep under 10MB for fast downloads
- **Naming:** Use lowercase with hyphens (e.g., my-paper.pdf)
